# Falafel Kingdom

A 2.5D platformer built with Vite, TypeScript, and Babylon.js.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Built app base path is `/falafelkingdom/`.

## Deploy

The built `dist/` folder is served as a subdirectory of the GitHub Pages site at `/falafelkingdom/`.

## Gameplay

- **Move**: A/D or Arrow keys
- **Jump**: Space / W / Up arrow (variable height - hold for higher jump)
- **Interact**: E key
- **Mobile**: Touch buttons shown at screen corners

### Objective

Help the sad Falafels by giving them Hummus! Collect Hummus by defeating Bullies and picking up coins. Give Hummus to Falafels to unlock paths and reach the goal flag.

### Level 1: Village of the Forgotten Falafels

- Find Hummus pickups on platforms
- Defeat/avoid the Bully patrolling the middle section
- Give 3 Hummus to the sad Falafel NPC to open the bridge
- Reach the goal at the end!

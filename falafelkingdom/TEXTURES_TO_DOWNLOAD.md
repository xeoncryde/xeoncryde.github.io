# Textures To Download

All textures below are **free** from [Poly Haven](https://polyhaven.com) (CC0 license).

The game works perfectly without these — they are optional enhancements only.

## Download Instructions

1. Visit the Poly Haven links above
2. Download at **2K resolution** (2048×2048) for good quality without huge file size
3. Save using the filenames mentioned in `FOLDER_STRUCTURE.txt`
4. Place in the correct subfolder

---

## Quick Start (just these 5 files)

If you only want a few textures, these make the biggest visual difference:

1. `sand_diffuse.jpg` → `/textures/sand/diffuse.jpg`
2. `stone_diffuse.jpg` → `/textures/stone/diffuse.jpg`  
3. `grass_diffuse.jpg` → `/textures/grass/diffuse.jpg`
4. `gold_diffuse.jpg` → `/textures/gold/diffuse.jpg`
5. `ice_diffuse.jpg` → `/textures/crystal/diffuse.jpg`
